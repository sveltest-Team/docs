ADB服务 #

adb提供了一些让我们进行对adb服务的一些操作这里在uiapp在也进行了对相对应的操作进行了高级封装。

启动ADB服务 #

python
import uiapp

app = uiapp.start()
app.start_server()

返回的是自己的实例对象、使用该方法可以启动adb服务

断开ADB服务 #

python
import uiapp

app = uiapp.start()
app.kill_server()

可以看到连接的设备全部断开连接了、证明adb服务已被关闭掉了
